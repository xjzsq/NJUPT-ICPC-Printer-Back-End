# 南邮ICPC校队代码打印平台 服务端

网页端见此仓库：[NJUPT-ICPC-Printer-Front-End: 南邮ICPC校队代码打印平台 网页端](https://github.com/xjzsq/NJUPT-ICPC-Printer-Front-End)

## 开始使用

克隆并安装依赖：

```bash
pip install pywin32
```

然后使用 `py 1.py`即可运行在 `http://localhost:6088` 。其他使用见网页端仓库的 readme。
